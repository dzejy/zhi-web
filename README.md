# Zhi导航 ❄️

一个带沉浸式雪花粒子系统的毛玻璃风格浏览器导航首页，单文件 HTML 实现，零依赖，双击即用。

## 核心功能

### 搜索引擎
- 内置 8 个搜索引擎：百度、秘塔AI、Google、必应、DuckDuckGo、B站、知乎、百度AI
- 搜索框右侧点击引擎名，右上方弹出 3x3 毛玻璃选择面板
- 支持自定义管理：添加、删除、修改搜索引擎名称和 URL
- 搜索引擎配置持久化保存在 localStorage

### 导航卡片
- 6 大分类 54 个预设常用网站（生活、社区、影音、资讯、工具、资源）
- 编辑模式下可修改任意链接的名称和地址，也可修改分类标题
- 自定义数据 localStorage 持久化，支持一键恢复默认

### 雪花粒子系统
- 180 个 Canvas 雪花粒子，带景深、随风摇摆、近大远小
- 物理积雪堆叠：雪花落在搜索框、卡片顶部会逐层向上堆积，无高度限制
- 积雪边缘圆滑处理，自然倾泻扩散效果
- 鼠标点击清除 60px 半径内积雪
- 三种积雪模式可切换：狂暴积雪（默认）/ 普通积雪 / 关闭积雪

### 专注模式（Z 按钮）
- 点击搜索框左侧 Z 按钮，隐藏导航卡片和时钟
- 搜索框 + Z 按钮整体放大 2 倍并平滑移至屏幕中央
- 切换时触发动态暴风雪（强度根据当前积雪量自适应）并清空所有积雪

### 互动特效
- 鼠标移动轨迹生成雪花拖尾，带微重力缓缓下落
- 鼠标点击产生雪花图标爆裂动效（旋转放大渐隐）
- 背景图 25 秒周期呼吸缩放动画

### 实时时钟
- 搜索框上方大字体时间显示，精确到秒，Nunito 圆润字体
- 日期显示星期和完整日期

### 设置面板
- 右上角齿轮按钮打开右侧抽屉式设置面板
- 编辑模式开关、积雪模式切换、暂停/恢复雪花、恢复默认链接

### 界面细节
- 左下角提示：Tips：雪太多的时候点击Z清理雪吧
- 右下角诗句：孤舟蓑笠翁，独钓寒江雪。
- 底部居中：@Dzejy — 2026 — Zhi导航（浮动动效）
- 响应式布局：3列 / 2列 / 1列自适应

## 使用方法

把 index.html 和 snow.jpg 放在同一目录下，浏览器直接打开 index.html 即可。无需 Node.js，无需构建工具。

## 技术栈

纯 HTML + CSS + JavaScript 单文件，Canvas 2D 粒子系统，CSS backdrop-filter 毛玻璃，localStorage 持久化，Google Fonts Nunito。
